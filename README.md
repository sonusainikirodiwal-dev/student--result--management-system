# Student Result Management System

A modular Python + SQL project to manage student records, classes, tests, and marks.

## Features
- Add classes and students to school and project
- Assign class tests and enter marks
- View student performance
- Organized into multiple Python files for clarity

## Technologies Used
- Python
- SQLite
- Flask (if web-based interface is used)

## File Structure
- mainFunction.py: Core logic
- savedFunction.py: Data saving functions
- Membership.py: User roles and access
- 1.py to 13.py: Individual modules for specific tasks

## How to Run
1. Install Python 3.x
2. Clone the repository
3. Run mainFunction.py or app.py (if Flask used)# student--result--management-system
